Boid 演算法在音訊輸入引導下於活動場地人流管理中的模擬與分析

2024/07/01 至 2025/02/28 NSTC 113-2813-C-035-135-E

本專題旨在研究如何運用 SoC FPGA 技術，實現音訊的視覺化處理與軟硬體協同設計，並以 Boids 演算法作為應用實例。研究透過接收音訊，並在 DE1-SoC 的 FPGA 上進行處理，藉此改變 boids （模擬群體）的行為參數，以模擬並觀察音訊對群體行為的影響，例如在緊急情況下利用聲音疏散人群。

  • 採用軟硬體協同設計，將音訊處理與 VGA 影像顯示等高速運算部分在 FPGA 上實現，而 Boid 演算法的控制邏輯則在 HPS ( ARM 處理器) 上實現。
  • 在FPGA端使用五個FIR帶通濾波器(band pass filter)對輸入音訊進行過濾，並將處理後的振幅或頻率等參數傳遞至 HPS ，用以即時調整 Boid 演算法中的分離、對齊、凝聚等規則。
  • 成功將Boid演算法的模擬結果，以 2D 及 3D 的方式呈現在 VGA 螢幕上，並可透過鍵盤控制 3D 畫面的旋轉。
  • 成果： 在音訊處理方面，使用 FPGA 實作比純軟體( C 語言)實作的速度快 51 倍。
  • 成果： 在繪圖效能方面，使用 DE1-SoC 進行繪圖的平均速度比使用 Windows 系統的Library快約 6 倍。

應用於多圖案微影技術之研究

2023/07/01 至 2024/02/29 NSTC 112-2813-C-035-080-E

隨著半導體追求先進製程，單次曝光的光刻技術已無法滿足精度需求，可能產生短路等瑕疵。為解決此問題，本研究著重於多重曝光技術，該技術將電路圖分割至不同圖層，分次曝光與蝕刻。計畫核心在於如何利用模擬退火演算法(Simulated Annealing)，將一個完整的電路圖進行有效的分層，以達到最佳化的分割。

  • 將電路圖中，彼此間距小於光刻機最小精度的元件抽象化為圖論中的節點 (Node)，並在這些節點之間建立連線，形成衝突圖 (Conflict Graph)。
  • 把圖層分割問題轉換為圖著色問題 (Graph Coloring Problem)，規定相鄰的元件節點必須塗上不同顏色，代表它們被分配到不同的曝光圖層。此方法利用了二部圖 (Bipartite Graph) 的概念進行驗證。
  • 採用模擬退火演算法，以「顏色密度」的均衡性作為評估函式 (Fitness Function)，來尋找全局最佳的塗色（分層）方案，從而提升分割效率。
  • 成果： 在 ICCAD 2015 Contest 提供的測試案例中，本研究所開發的演算法於 `iccad2015_input.case1` 達到了99.46的高分。
